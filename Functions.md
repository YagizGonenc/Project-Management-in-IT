# Core System Functions

## 1. Assignment Submission
**Function Name:** Submit Assignment  
**Input:** Student ID, assignment file  
**Process:** System validates file, stores it, and records timestamp  
**Output:** Confirmation message, stored submission  
**Feedback:** Student receives successful upload notification  
**KPIs:** Upload success rate, average upload time

---

## 2. Instructor Notification
**Function Name:** Notify Instructor  
**Input:** New assignment submission event  
**Process:** System sends automated alert to instructor  
**Output:** Email/push notification  
**Feedback:** Instructor acknowledges the alert  
**KPIs:** Notification delivery time, notification success rate

---

## 3. Feedback & Grading
**Function Name:** Provide Feedback  
**Input:** Instructor comments and grade  
**Process:** Instructor uploads evaluation through the portal  
**Output:** Stored feedback and grade  
**Feedback:** Student receives notification  
**KPIs:** Feedback turnaround time, grading accuracy rate

---

## 4. Student Review & Learning Update
**Function Name:** View Feedback  
**Input:** Student login and submission ID  
**Process:** System displays comments, grade, and revision notes  
**Output:** Student understanding and learning updates  
**Feedback:** Optional rework submission  
**KPIs:** Student feedback engagement rate, rework submission rate

---

## 5. Rework Submission (Optional)
**Function Name:** Submit Rework  
**Input:** Updated assignment  
**Process:** System stores revision and notifies instructor  
**Output:** Revised version  
**Feedback:** Instructor reviews again  
**KPIs:** Rework quality improvements, rework cycle time
